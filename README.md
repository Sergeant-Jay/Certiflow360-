Certiflow360 – Fleet Intelligence Prototype
Overview
Certiflow360 is a frontend prototype of a fleet intelligence platform designed for logistics operations across Southern Africa. The prototype focuses on usability, responsiveness, and clear presentation of operational data, demonstrating how real‑time fleet monitoring, risk management, and cost optimisation could be presented in a production system.
This project was developed as part of an occupational experiential learning programme.

Features

Responsive single‑page website
Live operations dashboard (mock data)
Risk & safety view
Cost & fuel savings breakdown
ROI calculator
Theme selector for improved readability
Mobile‑friendly layout and interactions
Contact / Book a Demo section


Technologies Used

HTML5
CSS3
JavaScript
GitHub Pages (deployment)


Mock Data
The prototype uses mock data to simulate:

Active fleet statistics
SLA performance
Alerts and live feed updates
Monthly savings and cost distribution

No real production data is stored or processed.
